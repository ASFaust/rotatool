# Rota Tool

This project is a web application for generating rotas for the field projects of archelon.

The shifts covered here are:

- Morning Surveys
- Night Surveys
- Shading
- Excavation
- Public Excavation
- Hatchery
- Kiosk
- Information Table
- Slideshows
- Beach Walks
- Driver Shifts
- Basecamp Shifts
- Custom Shifts
